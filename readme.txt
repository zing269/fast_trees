Fast Trees
==========

Changes sapling node timers to 1 to 30 minutes. 

Configure by setting the fast_trees_min_time and fast_trees_max_time values in minetest.conf.

fast_trees_min_time = 60
fast_trees_max_time = 3600

Download: https://github.com/zing269/fast_trees/archive/master.zip
Source: https://github.com/zing269/fast_trees
